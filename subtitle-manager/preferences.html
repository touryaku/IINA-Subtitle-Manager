Subtitle Manager Preferences

Behavior

Show OSD
Errors are still shown even if OSD is disabled.
Auto-hide Window
This corresponds to the plugin menu option "Auto-hide Window in Fullscreen".

Shortcuts

Show Sidebar
Example: Meta+Alt+s (Cmd+Option+S)
Show Window
Example: Meta+Alt+e (Cmd+Option+E)
Copy Timestamped
Default is Meta+Alt+c (Cmd+Option+C) to avoid conflicts with Edit > Copy. If you want Cmd+C, use macOS App Shortcuts to override (recommended).
Apply Changes
Shortcut changes apply after reloading the plugin (IINA: Plugins > Reload All Plugins) or restarting IINA.

Notes

Key string format is the same as IINA plugin menu keyBinding values. Common modifiers: Meta (Cmd), Alt (Option), Shift, Ctrl. Example: Meta+Alt+e.
macOS App Shortcuts: System Settings > Keyboard > Keyboard Shortcuts > App Shortcuts. Use the exact menu item title (e.g. "Show Subtitle Manager Window").
